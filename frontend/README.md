# Frontend (React)

## Overview

This directory contains the frontend application built using **React**.

The frontend provides a user interface for interacting with the retail management system.

---

# Features

* login interface
* dashboard view
* order management
* inventory display
* payment tracking

---

# Responsibilities

* collect user input
* send requests to backend API
* display data returned from the backend

---

# Structure

```text
components/ → reusable UI components
pages/      → main views
services/   → API calls
context/    → state management
```
